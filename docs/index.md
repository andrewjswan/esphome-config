# ESPHome Config
My configurations of different devices based on ESPHome


<div class="grid cards" markdown>

- :material-blur: __[Air Freesh]__ – Automatic air freshener, uses AirWick or Glade as a donor, battery operated, integrated into Home Assistant via MQTT.
- :material-ghost: __[DAIKIN Air Sensor]__ – Daikin air sensor is BRY88AB151K (or BRY88AA151K).
- :material-led-strip-variant: __[Music Leds]__ – [A1-SLWF-03](https://smlight.tech/product/slwf-03/) WLED `Sound Reactive` LED Controller is a tiny adressable led strip controlled based on ESP32.
- :material-power-socket-de: __[Nous A1 Socket]__ – Config based on ESPHome [Nous A1](https://devices.esphome.io/devices/Nous-A1).
- :material-power-socket-de: __[Nous A8 Socket]__ – Config based on ESPHome [Nous A8T](https://devices.esphome.io/devices/Nous-A8T)
- :material-molecule-co2: __[Senseair S8 Light]__ – [SenseAir CO2 Sensor](https://esphome.io/components/sensor/senseair/) Lamp.
- :material-desk-lamp: __[Sonoff Mini Desk Lamp]__ – Config based on ESPHome [Sonoff Mini Relay](https://devices.esphome.io/devices/Sonoff-Mini-Relay).
- :material-power-plug-battery-outline: __[Mini (DC) UPS]__ – Mini (DC) UPS - ADC voltmeter monitoring throughput voltage.
- :material-lightbulb-on-outline: __[SvitloBot]__ – ESPHome firmware for SvitloBot | svitlobot.in.ua.
- :material-lava-lamp: __[Matrix Lamp]__ – A compilation of firmware adaptations based on the original idea of Gyver Lamp, for ESPHome.
- :material-clock-digital: __[Pixel Clock]__ – ESPHome firmware for Ulanzi TC001 Clock using [EspHoMaTriX](https://github.com/lubeda/EspHoMaTriXv2) component.

</div>
